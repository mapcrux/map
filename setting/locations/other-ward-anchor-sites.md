# Other Ward Anchor Sites (Unreached)

## Type
- Magical anchor points

## Description
- There are twelve ward anchors in total, and only one has been restored.
- Unreached anchor sites are believed to include:
  - Orcish lands
  - Deep forest sanctuaries
  - Mountain caverns
  - Coastal ruins

## Relative Location
- Spread across the continent in a circular pattern.
